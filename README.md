# 🚀 Community Frontend

> FastAPI 기반 백엔드 API와 연동되는 **커뮤니티 서비스 프론트엔드 프로젝트**

---

## 📌 프로젝트 소개

Community Frontend는  
FastAPI 기반 백엔드 API와 연동되는  
**회원관리 및 게시판 기능을 제공하는 웹 프론트엔드 애플리케이션**입니다.

본 프로젝트는 다음을 목표로 합니다:

- 사용자 친화적인 UI/UX 제공
- RESTful API 기반 백엔드와의 데이터 통신
- 실제 서비스 수준의 프론트엔드 구조 설계

👉 Backend Repository  
https://github.com/Tae-Geon-Kim/FastAPI-Community-Backend

---

## 🛠️ 기술 스택

### Frontend
- Next.js
- React
- JavaScript (또는 TypeScript)
- CSS / TailwindCSS (선택)

### Backend (연동)
- FastAPI
- PostgreSQL
- JWT 인증 (예정)

## ✨ 주요 기능

### 👤 회원 기능
- 회원가입
- 로그인

### 📝 게시판 기능
- 게시글 작성
- 게시글 조회 (목록 / 상세)
- 게시글 수정 / 삭제

### 🔐 인증 (예정)
- JWT 기반 로그인 인증
- 토큰 저장 및 인증 처리
